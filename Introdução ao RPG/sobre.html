Descrição do Projeto

Este projeto visa criar uma plataforma interativa para amantes de RPG (Role-Playing Game), onde os usuários podem aprender sobre as diferentes classes de RPG, criar fichas de personagens e participar de aventuras épicas em um ambiente virtual. Nossa intenção é proporcionar uma experiência rica e envolvente, que incentive a criatividade e a colaboração entre os jogadores.

A plataforma contará com várias funcionalidades, incluindo a descrição detalhada das classes de RPG, um gerador de fichas e um espaço para os jogadores compartilharem suas histórias e aventuras. Estamos comprometidos em criar um espaço seguro e divertido para todos os entusiastas de RPG.
